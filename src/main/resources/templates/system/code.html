화면 설명서(목록탭)

1. 상위분류명을 통해 검색하여 조회할 수 있습니다.

2. 신규로 상위코드 등록 할 경우 코드, 코드명, 설명을 입력하고 저장 버튼을 누릅니다.

3. 중위코드 생성 시 상위분류명을 클릭 후 하위추가 버튼을 누르고 입력값 입력 후 저장 버튼을 누릅니다.

3-1. 하위코드 생성 시 증위분류명을 클릭 후 하위추가 버튼을 누르고 입력값 입력 후 저장 버튼을 누릅니다.

4. 상위분류명, 중분류명, 하위분류명 중 삭제 대상을 클릭 후 삭제 버튼을 누릅니다.

※ 상위분류명 선택 후 삭제 시 해당 전체 데이터가 삭제 됩니다.
